# ToIPLabs

A series of labs designed to create understanding about Hyperledger's Trust over IP (ToIP) technology stack. Finally, a trust layer for the Internet!

## Run CMD

First run (will run up after building)
```bash
$ sh run.sh
```

Up // Down
```bash
$ sh up.sh
$ sh down.sh
```
